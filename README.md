# PeerTutorFinder — Empowering students to learn, share, and grow together.....

**PeerTutorFinder** is a simple web app prototype that helps students find peer tutors within their campus.  
It connects learners with helpful peers or seniors for study support, using **Firebase** for authentication and database, and **Gemini AI** to match doubts with the best available tutor.

---

## ✨ **Features**

✅ Secure login and registration (Firebase Auth)  
✅ Choose your role: Learner or Tutor  
✅ Learners can post a doubt/question  
✅ Gemini AI suggests best matching tutors based on subjects and availability  
✅ Firestore stores user profiles, doubts, and matches

---

## 🚀 **Tech Stack**

- **Frontend:** React / Next.js (built in Google IDX)
- **Backend:** Firebase Auth + Firestore
- **AI:** Google Gemini API / Vertex AI
- **Deployment:** IDX or Firebase Hosting

---

## 🏫 **How it helps**

- Empowers students to find help fast
- Encourages peer-to-peer learning in campus
- Helps seniors share knowledge and build mentoring culture
- Can be scaled for schools, colleges, coaching centres

## ⚙️ **Setup**

1️⃣ **Clone the repository**
```bash
git clone https://github.com/Aditya-10-Singh/PeerTutorFinder.git
cd PeerTutorFinder
```

2️⃣ Install dependencies
```bash
npm install
```

3️⃣ Add your Firebase config

- Create a firebase.js file in the project root.
- Add your Firebase apiKey, authDomain, projectId and other details.

4️⃣ Run the project
```bash
npm run dev
```
